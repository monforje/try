# Balanced News — App

Мобильное приложение на React Native (Expo).

## Запуск

```bash
npm install
npm start
```

Перед запуском приложение читает настройки из `.env` (например `API_BASE_URL`).

## Основные экраны
- Splash
- Quiz
- Feed (лента)
- Article (статья)

## Хранилище
- AsyncStorage: bias-вектор, отложенные реакции
